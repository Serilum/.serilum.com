Serilum Mod Translations

All translation files are parsed by the Collective library mod, so dependent mods don't have to manage them separately. Non-English languages are downloaded on runtime in-game once, then cached. I've done it this way so Collective's file size doesn't increase by ~3Mb with all the translation data. Changes made to the language files on GitHub will automatically be pulled into the game.

To submit translations/improvements please pick a language on the left, fill in what you can, then copy the file and open a pull request. You'll need a GitHub account for this.

Anything you leave blank simply falls back to English in-game, so partial translations are also very welcome.

Thank you for contributing and giving the mod collection a try!
